## 介绍一下

博主从2018年因为业务原因开始接触图数据库，在金融公司做`金融欺诈风控`相关业务，写该系列文章的目的主要是在图库使用中深感图库的资料较少，希望可以为图数据库社区做一些微薄的贡献，与君共勉

主要使用图库：**JanusGraph**

了解图库：**Neo4j**、**Nebula Graph**
